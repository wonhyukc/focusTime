포모도로 타이머 샘플 데이터 생성

이 페이지는 포모도로 타이머 앱의 통계 기능을 테스트하기 위한 다양한 샘플 데이터를 생성합니다. 생성된 데이터는 CSV 파일로 다운로드됩니다.

처리 중...

생성되는 데이터 특징:

  • 1년 치 데이터 (현재 날짜로부터 1년 전까지)
  • 다양한 프로젝트 (9개 프로젝트)
  • 다양한 집중 세션 시간 (15분, 25분, 30분, 45분, 60분)
  • 요일별 사용 패턴 반영 (주중에 더 많은 사용량)
  • 시간대별 사용 패턴 반영 (오전 9-12시, 오후 2-6시에 집중)
  • 계절별/월별 사용 패턴 반영 (시험 기간에 사용량 증가)
  • 집중 세션과 휴식 세션 모두 포함

CSV 파일 형식:

  • 시작시각(년월일시분): 세션 시작 시간 (YYYY-MM-DD HH:MM 형식)
  • 세션: 세션 유형 (focus, shortBreak, longBreak)
  • 지속시간: 세션 지속 시간 (분)
  • 프로젝트: 프로젝트 이름
